# Online Hospital Appointment System

This system allows patients to book medical appointments online and check their appointment details anytime. It helps reduce hospital crowding and makes scheduling easier for both patients and medical staff, with 24/7 access for users.

---

### 1. Project Impact (Results)
I designed this system to solve the problem of long wait times and messy paper schedules.

| Current Challenge | Digital Solution | Result for Hospital |
| :--- | :--- | :--- |
| **Crowded Waiting Rooms** | Online Booking System | Fewer people waiting in the hospital |
| **Double Bookings** | Conflict Prevention Logic | No more overlapping appointments |
| **Hard to Change Data** | Staff-Friendly Dashboard | Nurses can update schedules instantly |
| **Manual Tracking** | Real-time Data Sync | Accurate and up-to-date patient lists |

---

### 2. How the System Works (Simple Design)
The system uses modern tools to ensure it is fast, reliable, and easy to use on any device.

| Part | Tool Used | Why it matters |
| :--- | :--- | :--- |
| **User Interface** | Next.js | Provides a smooth and fast experience for patients |
| **Data Storage** | Google Sheets API | Easy for hospital staff to manage without tech training |
| **Validation** | Server-side Checks | Ensures all patient data is correct before saving |
| **Speed** | Vercel Edge Runtime | The website loads instantly, even during busy times |

---

### 3. Safety and Trust (Governance)
Patient privacy is the top priority. The system is built following professional healthcare standards.

| Feature | Implementation Detail |
| :--- | :--- |
| **Expert Design** | Requirements were defined by Senior Professional Nurses |
| **Privacy Laws** | Strictly follows Thailand's PDPA rules for patient data |
| **Security** | Sensitive information is processed only on secure servers |
| **Data Ethics** | Data is used only for medical appointments, never for other purposes |
